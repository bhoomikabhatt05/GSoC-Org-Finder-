## ✨ Overview

GSoC 2026 Org Finder is a fast and beginner-friendly platform designed to help students explore all selected Google Summer of Code organizations in one place.

Instead of manually browsing hundreds of organizations, this tool helps contributors:

* discover organizations by domain or language
* compare organizations side-by-side
* find beginner-friendly issues
* analyze competition level
* explore project ideas quickly

Built especially for:

* GSoC beginners
* GSSoC contributors
* open-source newcomers
* students exploring organizations efficiently

## 🚀 Beginner Quick Start

1. Fork the Repository

Click the Fork button on GitHub.

2. Clone Your Fork

git clone https://github.com/your-username/GSoC-Org-Finder-.git
cd GSoC-Org-Finder-

3. Install Dependencies

npm install

4. Start Development Server

npm run dev

5. Create a Branch

git checkout -b feature-name

6. Make Changes

You can:

* improve UI
* add organizations
* fix bugs
* improve documentation
* add filters/features

7. Commit Changes

git add .
git commit -m "Improve README onboarding experience"

8. Push Changes

git push origin feature-name

9. Open a Pull Request

Create a PR and wait for mentor review.

## 📚 Features

🔍 Smart Organization Discovery

* Full-text organization search
* Domain filtering
* Language filtering
* Multi-tag filtering
* Competition-level filtering

⚖️ Organization Comparison

Compare up to 3 organizations side-by-side using:

* stars
* forks
* activity
* competition level
* GitHub issues
* technologies

🟢 Good First Issues

Dedicated beginner-friendly issue explorer with:

* live GitHub issue fetching
* labels
* comment counts
* newest-first sorting

📈 Live GitHub Stats

* Stars
* Forks
* Open issues
* Last commit activity
* Repository activity levels

🌙 Modern UI

* Dark mode
* Responsive design
* Mobile-friendly layouts
